Solve nonlinear equation by numerical methods

Dichotomy method

Solve $f(x)=0$, where $x\in \mathbb{R}$ and $f$ is a continuous function. With initial interval $[a_0,b_0]$, dichotomy method is as follows, for $k=0,1,\cdots$, $$ \begin{array}{l l} c_k&=\frac{a_k+b_k}{2}\\ if & f(a_k)f(c_k)>0\\ & a_{k+1}=c_{k}, b_{k+1}=b_k\\ else &b_{k+1}=c_k, a_{k+1}=a_{k} \end{array} $$

Build solve.py 

def fun(x):return(x**3/24+x/2+1/(x**2+1)/2-2)
    # define a function
init=[-4.3,4.3]
def sv_dc(f,init,it=10):
    # dichotomy method
    a,b=init
    k=0
    while k < it:
        k=k+1
        c=(a+b)/2
        if f(a)*f(c)>0:
            a=c
        else:
            b=c
    rt=(a+b)/2
    return(rt)
it=7
rtl=sv_dc(fun,init,it)

Newton method

Solve $f(x)=0$, where $x\in \mathbb{R}$ and $f$ is a differentiable function. $f'(x)$ is denoted as the derivative of $f(x)$. With initial $x_0$, Newton method is as follows, for $k=0,1,\cdots$ $$x_{k+1}=x_k-\frac{f(x_k)}{f'(x_k)}$$

Build solve.py 

def fun(x):return(x**3/12+x+1/(x**2+1))
    # define a function
def drv(x):return(x**2/12*3+1-2*x/(x**2+1)**2)
    # define the derivate of fun
def sv_nt(f,d,init,it=10):
    # Newton method
    k=0
    rt=init
    while k < it:
        k=k+1
        rt=rt-f(rt)/d(rt)
    return(rt)
init=4.1
it=7
rt=sv_nt(fun,drv,init,it)
